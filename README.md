# 📚 Google Sheets Linked Quiz System

GoogleスプレッドシートをCMS（管理画面）として活用した、学習用クイズアプリケーションです。
塾の生徒様が、PCやスマホからいつでも手軽に英単語学習を行える環境を目指して構築しました。

## 🌐 Live Demo
**[アプリを体験する](https://quiz-app-six-teal-44.vercel.app/)**
※バックエンドはRender（無料プラン）を使用しているため、最初の読み込みに15〜30秒ほどかかる場合があります。

## 📝 プロジェクト開発の背景
従来はGoogleフォームでクイズが運用されていましたが、「より学習に特化した専用のユーザーインターフェース（UI）を提供したい」というご要望を受け、本アプリの開発に至りました。 すでにGoogleのエコシステム（スプレッドシート等）を日常的に使用されていたため、開発にあたっては**「使い慣れたツールを管理画面（CMS）としてそのまま使い続けられること」**を最優先のコンセプトとしました。

## 📝 こだわった工夫と設計思想
- **「動的なカテゴリ生成」による完全メンテナンスフリー化 当初は特定のシート名のみを読み込む設定でしたが、「今後、講師側で自由に問題カテゴリ（シート）を増やしたい」というニーズを想定し、スプレッドシートのタブ名を自動検知してメニューに反映するロジックを実装しました。これにより、新しいカテゴリの追加から問題の更新まで、非エンジニアの講師の方のみで完結できる運用体制を実現しました。**
- **実用性の重視とAPIの抽象化 複雑な認証やスプレッドシートAPIの操作をバックエンド（Node.js）側で隠蔽し、フロントエンドからはクリーンなAPIを叩くだけでデータを取得できる構造にしました。これにより、将来的な機能拡張もしやすい保守性の高い設計となっています。**
- **本番環境を見据えた環境管理 .envファイルと環境変数を活用し、ローカル開発環境と本番環境（Vercel/Render）の接続先を自動的に切り替える仕組みを構築。手動によるURLの書き換えミスを排除し、安全かつ迅速なデプロイフローを確立しました。**

## 🛠 使用技術

### Frontend
- **Vue.js 3 (Composition API)**
- **Vite** (ビルドツール)
- **Tailwind CSS** (スタイリング)
- **Vercel** (ホスティング)

### Backend
- **Node.js / Express**
- **Google Sheets API v4**
- **Render** (ホスティング)

### Database
- **Google Sheets** (スプレッドシート)

## ✨ 主な機能
1. **スプレッドシート同期**: タブ名が自動的にクイズカテゴリとしてメニューに反映されます。
2. **レスポンシブデザイン**: スマートフォン、タブレット、PCのすべてのデバイスに最適化。
3. **環境切り替え**: `.env`と環境変数を活用し、ローカル開発環境と本番環境の接続先を自動切り替え。
4. **堅牢なエラーハンドリング**: API通信のタイムアウト制御や、認証エラーの適切な表示を実装。
